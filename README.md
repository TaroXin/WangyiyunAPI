## 网易云 API

Step 1 :
```
npm install
```

Step 2 :
```
node index
```

Step 3 :
```
http://localhost:3000/recommendLst
http://localhost:3000/playlist/:playlistId
http://localhost:3000/song_list/:playlistId
```
